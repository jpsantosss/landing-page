Jota Bot

O bot perfeito para moderação de seu servidor e de um jeito fácil.

O que ele faz

1. Cria um sistema de captcha para novos usuários para garantir que não são robôs.

2. Banimento automático para aqueles que infrigirem as regras do seu servidor.

3. Possui um sistema de aviso para link suspeitos enviados na comunidade.

4. Adiciona comando de música para você e seus amigos curtirem enquanto conversam.
